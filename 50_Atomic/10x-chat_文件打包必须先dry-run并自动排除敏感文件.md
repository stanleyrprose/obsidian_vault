# 文件打包给模型前，必须可审计（dry-run）且自动排除敏感文件

## 定义
把文件内容作为上下文发送给模型前，要支持预览（dry-run）并排除常见敏感文件。

## 本质
将“上下文注入”变成可控流程，降低泄露风险。

## 解决的问题
- 防止误把 `.env`、私钥、token 等打包发送
- 让用户在发送前能看见真实会被发送的内容

## 边界
- 仍需要用户自觉检查（dry-run 不等于绝对安全）
